# 💳 Cartão Animado (Card Hover Animation)

Projeto front-end que simula um cartão de crédito com efeito de rotação 3D ao passar o mouse (hover). 
Ideal para praticar animações e conceitos de CSS.

---

## 📸 Preview

O projeto apresenta:

- Frente do cartão com número, nome e validade
- Verso do cartão com tarja magnética e assinatura
- Animação 3D ao passar o mouse

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Google Fonts (Poppins)

---

## 🎯 Funcionalidades

- Animação de rotação 3D no hover
- Design moderno com efeito glassmorphism
- Estrutura simples e organizada
- Uso de perspectiva para efeito realista

## ⚙️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/devtiagobr/card-animation.git
```

2. Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

3. Abra o arquivo `index.html` no navegador

---

## 🧠 Conceitos aplicados

- `transform: rotateY()` para rotação do cartão
- `transform-style: preserve-3d` para manter o efeito 3D
- `backface-visibility: hidden` para esconder o verso
- `perspective` para profundidade
- `backdrop-filter` para efeito de vidro
